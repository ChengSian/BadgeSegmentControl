## Requirements

* Xcode 8 
* iOS 9.0+ target deployment
* Swift 3 project 

## Installation

BadgeSegmentControl is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "BadgeSegmentControl"
```
